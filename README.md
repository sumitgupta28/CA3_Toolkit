
# CA3 Marks Processing ‚Äî Setup & Usage Guide


## Initial setup

Run the one-time setup script for your operating system. It will:
- Verify Python 3 is installed
- Create a `.venv` virtual environment
- Upgrade `pip`
- Install all dependencies from `requirements.txt`

**macOS / Linux:**
```bash
chmod +x setup.sh   # only needed once
./setup.sh
```

**Windows:**
```bat
setup.bat
```

After setup completes, activate the virtual environment before running any scripts:

```bash
# macOS / Linux
source .venv/bin/activate

# Windows
.venv\Scripts\activate
```

> **PDF conversion** uses `docx2pdf` (requires Microsoft Word on macOS/Windows).
> LibreOffice is also supported ‚Äî if installed, it takes priority over `docx2pdf`.

## Generating test documents (optional)

Before running the full workflow, you can generate 10 sample student `.docx` files to test the pipeline end-to-end:

```bash
python generate_test_documents.py
```

This creates pre-filled student documents in the `students/` folder using a CA3 template (`Seal_Top_Sheet_template.docx`). Each document simulates a student who has completed the top information section and signed the bottom.

Edit the constants at the top of `generate_test_documents.py` to customise:
- `TEMPLATE_PATH` ‚Äî path to your blank CA3 template
- `COMMON_DATA` ‚Äî shared fields (college, subject, teacher, etc.)
- `STUDENTS` ‚Äî individual student details (name, 10-digit roll number, etc.)

A `.zip` archive of all generated files (`students_test_documents.zip`) is also created by default.

## Step-by-step workflow

### Step 1 ‚Äî Share template with students
Send the original `.docx` template to students.
They fill in the top section (Program Name, Subject, Name, Roll Number, etc.)
and sign at the bottom. **Each student must have a unique 10-digit Roll Number.**

### Step 2 ‚Äî Collect returned documents
Place all returned `.docx` files into the `students/` folder.

### Step 3 ‚Äî Generate the marks spreadsheet
```bash
python extract_to_excel.py
```
This reads every `.docx` in `students/` and creates `CA3_Marks/marks.xlsx` with:
- One row per student
- Student info pre-filled
- **Yellow cells** = Marks Awarded + Feedback columns (you fill these in)

### Step 4 ‚Äî Fill in marks
Open `CA3_Marks/marks.xlsx` and fill in the **yellow columns**:
- `Awarded 1A` through `Awarded 7` ‚Äî marks for each question
- `Strengths` ‚Äî examiner feedback
- `Areas for Improvement`
- `Corrective Measures`

Save and close the Excel file.

> **Testing shortcut** ‚Äî instead of filling the spreadsheet manually, run:
> ```bash
> python fill_dummy_marks.py
> ```
> This populates all yellow cells with randomly generated marks (60‚Äì100% of
> each question's allotted marks) and sample feedback text so you can proceed
> straight to Step 5.

### Step 5 ‚Äî Update documents and export PDFs
Place your signature image as `examiner_signature.png` in the project folder, then run:
```bash
python fill_marks_and_export.py
```
This will:
- Match each `.docx` to its Excel row via Roll Number (10-digit)
- Fill in Marks Awarded for each question
- Fill in Examiner Feedback
- Insert your signature image + today's date
- Save updated `.docx` to `output/docx/`
- Convert each document to PDF in `CA3_Marks_Pdf/`

## Cleanup

To delete all generated files and student documents (keeps `CA3_Marks/marks.xlsx` unless you choose to delete it):

**macOS/Linux:**
```bash
./cleanup.sh
```

**Windows:**
```bat
cleanup.bat
```

Or using Python (cross-platform, no prompt for marks.xlsx):
```bash
python cleanup.py
```

## Tips

- **Roll Number must match exactly** between the student's document and the Excel.
  If a student leaves Roll Number blank or enters it incorrectly, that file will be skipped.
- **Original files are never modified.** All output goes to `output/docx/` and `CA3_Marks_Pdf/`.
- If neither LibreOffice nor `docx2pdf` is available, `.docx` files are still updated ‚Äî only PDF export is skipped.

## Troubleshooting

| Problem | Fix |
|---------|-----|
| `ModuleNotFoundError: docx` | Run `pip install python-docx openpyxl docx2pdf` |
| Roll Number not found in Excel | Check that the 10-digit Roll Number in the `.docx` matches the Roll Number column in Excel |
| PDF not generated | Ensure `docx2pdf` is installed (`pip install docx2pdf`) and Microsoft Word is available |
| Signature not appearing | Ensure `examiner_signature.png` is in the same folder as the scripts |
| `CA3_Marks/marks.xlsx` not found | Run `extract_to_excel.py` first to generate it |
