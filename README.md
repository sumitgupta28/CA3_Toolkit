# CA3 Marks Processing — Setup & Usage Guide

## What you need installed

```bash
# Python packages
pip install python-docx openpyxl docx2pdf
```

> **PDF conversion** uses `docx2pdf` (requires Microsoft Word on macOS/Windows).
> LibreOffice is also supported — if installed, it takes priority over `docx2pdf`.

---

## Folder structure

Set up your project folder like this before starting:

```
your-project/
│
├── students/                   ← Put all student .docx files here
│   ├── student_U001.docx
│   ├── student_U002.docx
│   └── ...
│
├── CA3_Marks/
│   └── marks.xlsx              ← Auto-generated by extract_to_excel.py
│
├── CA3_Marks_Pdf/              ← Final PDFs (auto-created)
│
├── examiner_signature.png      ← Your signature image (PNG recommended)
│
├── extract_to_excel.py         ← Run first
├── fill_marks_and_export.py    ← Run after filling Excel
│
└── output/
    └── docx/                   ← Updated .docx files (auto-created)
```

---

## Step-by-step workflow

### Step 1 — Share template with students
Send the original `.docx` template to students.
They fill in the top section (Program Name, Subject, UPID, Name, Roll No, etc.)
and sign at the bottom. **Each student must have a unique UPID.**

### Step 2 — Collect returned documents
Place all returned `.docx` files into the `students/` folder.

### Step 3 — Generate the marks spreadsheet
```bash
python extract_to_excel.py
```
This reads every `.docx` in `students/` and creates `CA3_Marks/marks.xlsx` with:
- One row per student
- Student info pre-filled
- **Yellow cells** = Marks Awarded + Feedback columns (you fill these in)

### Step 4 — Fill in marks
Open `CA3_Marks/marks.xlsx` and fill in the **yellow columns**:
- `Awarded 1A` through `Awarded 7` — marks for each question
- `Strengths` — examiner feedback
- `Areas for Improvement`
- `Corrective Measures`

Save and close the Excel file.

### Step 5 — Update documents and export PDFs
Place your signature image as `examiner_signature.png` in the project folder, then run:
```bash
python fill_marks_and_export.py
```
This will:
- Match each `.docx` to its Excel row via UPID
- Fill in Marks Awarded for each question
- Fill in Examiner Feedback
- Insert your signature image + today's date
- Save updated `.docx` to `output/docx/`
- Convert each document to PDF in `CA3_Marks_Pdf/`

## Cleanup

To delete all generated files and student documents (keeps `CA3_Marks/marks.xlsx` unless you choose to delete it):

**macOS/Linux:**
```bash
./cleanup.sh
```

**Windows:**
```bat
cleanup.bat
```

Or using Python (cross-platform, no prompt for marks.xlsx):
```bash
python cleanup.py
```

---

## Tips

- **UPID must match exactly** between the student's document and the Excel.
  If a student leaves UPID blank or misspells it, that file will be skipped.
- **Original files are never modified.** All output goes to `output/docx/` and `CA3_Marks_Pdf/`.
- If neither LibreOffice nor `docx2pdf` is available, `.docx` files are still updated — only PDF export is skipped.

## Troubleshooting

| Problem | Fix |
|---------|-----|
| `ModuleNotFoundError: docx` | Run `pip install python-docx openpyxl docx2pdf` |
| UPID not found in Excel | Check spelling in both the .docx and the Excel UPID column |
| PDF not generated | Ensure `docx2pdf` is installed (`pip install docx2pdf`) and Microsoft Word is available |
| Signature not appearing | Ensure `examiner_signature.png` is in the same folder as the scripts |
| `CA3_Marks/marks.xlsx` not found | Run `extract_to_excel.py` first to generate it |
